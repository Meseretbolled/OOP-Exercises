OOP(object oriented programming) 
Object-Oriented Programming (OOP) is a programming paradigm that organizes code around objects and their interactions. 
It focuses on modeling real-world entities as objects and provides a set of principles and concepts to structure and design software systems.
OOP promotes 
     modularity
     reusability
     maintainability of code.
